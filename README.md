# helix-themes

## Usage

Download toml file to ~/.config/helix/themes
```
$ wget https://raw.githubusercontent.com/tkmfujise/helix-themes/main/gessetti.toml -O ~/.config/helix/themes/gessetti.toml
```

edit config
```
$ vi ~/.config/helix/config.toml
```
```
theme = "gessetti"
```
